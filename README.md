# sporttsotron Section 1: Title and Mission
Sporttsotron 🏆
Strip away the non-stop noise and betting promotions.

🎯 Mission Statement
In a world where sports coverage is increasingly dominated by commercial partnerships, heavy commercialization, and betting odds, the essence of the game can easily get lost. Sporttsotron is a lightweight developer tool that uses the Gemini API to cut through the clutter—delivering pure, historical-focused narratives of game match-ups so fans can enjoy the pure statistics and history of the sport without the distractions.

Section 2: How It Works
🏛️ How It Works
Sporttsotron is a single-page web utility with a simple pipeline:
[Team Matchup Input]
         │
         ▼
[Gemini API — guided by a specialized system prompt]
 ──► Standardizes match-up statistics and scores
 ──► Builds an explicit historical context
 ──► Isolates record-breaking moments into structured highlights
         │
         ▼
[Optimized Narrative Output] ──► Ready to display the pure story of the game

Section 3: Pillars, Tech Stack, and License
​✨ Key Pillars
​LLM-Guided Narratives: A specialized system prompt directs Gemini to synthesize comprehensive scores and statistics, highlighting what truly happened on the field.
​Record-Breaking Alerts: Special call-outs ensure that any historical feats or record-breaking moments during a game are immediately apparent to the user.
​Data Visualization: Incorporates visual summaries to present complex statistics and metrics in an easy-to-read, pleasing format.
​BYOK (Bring Your Own Key): Users supply their own Gemini API key, which is securely stored only in local browser storage and never sent to any third-party servers.
​🛠️ Tech Stack
​Front-End: Vanilla JavaScript, HTML5, CSS3
​AI Backend: Gemini API (gemini-2.5-flash)
​Deployment: GitHub Pages (fully static, serverless)
